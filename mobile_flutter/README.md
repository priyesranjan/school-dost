# ERP School Mobile

Flutter Android app for the ERP School platform.

## Included in this phase

- tenant-aware login
- persisted session
- dashboard metrics
- assignments
- classroom resources
- fee structures
- timetable

## Local run

```bash
cd mobile_flutter
flutter pub get
flutter run
```

For the Android emulator, use `http://10.0.2.2:4000/api` as the backend URL.
